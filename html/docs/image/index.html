# 图像标签

# 基础图像标签

  • img：通过 loading 属性指定指定图片加载的行为。
    • auto：浏览器的默认行为。
    • lazy：启用懒加载。
    • eager：立即加载，无论它在页面那个位置。
  • figure：图像区块，将图像和相关信息封装在一起。
  • figcaption：figure 的可选子元素，表示图像的文本描述。

# 响应式图片

  • img 标签：

    • srcset 属性，指定多张图像，适应不同像素密度的屏幕。
    <img srcset="foo-320w.jpg,
         foo-480w.jpg 1.5x,
         foo-640w.jpg 2x"
 src="foo-640w.jpg">
