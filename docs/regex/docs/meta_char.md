# 元字符

| 字符 | 说明                         |
| ---- | ---------------------------- |
| `.`  | 匹配除了换行符之外的任意字符 |
| `\w` | 匹配字母或数字或下划线或汉字 |
| `\s` | 匹配任意的空白字符           |
| `\d` | 匹配数字                     |
| `\b` | 匹配单词的开始或结束         |
| `^`  | 匹配字符串的开始             |
| `$`  | 匹配字符串的结束             |
