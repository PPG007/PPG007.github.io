# GitLab 扩展语法

- 用户 Mention：

  在需要的地方直接 `@${用户名}` 即可。

- Issue 引用：

  使用 `#${Issue ID}` 即可完成 Issue 引用，井号后面没有空格而且只能有一个井号。

- Milestone （里程碑）引用：

  使用 `%${id}` 完成 Milestone 引用。

- Merge Request 的引用：

  使用 `!${Issue ID}`，注意叹号是半角。

- Commit 引用：

  复制 Commit ID 直接粘贴即可。

- Emoji：

  使用半角冒号后面接表情代号即可。

- 任务列表：

  ```markdown
  - [ ] item 1.
  - [x] item 2.
  ```
